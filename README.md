# Control de Bebé

App Flutter para el seguimiento del bebé: peso, alimentación (pecho/biberón) y pañales.

## Características

- **Peso**: Registro de peso con gráfica de evolución y comparación con percentiles OMS
- **Alimentación**: Cronómetro de lactancia (teta izquierda/derecha) y registro de biberón
- **Pañales**: Control de cambios (mojado, sucio, ambos)
- **Home**: Panel resumen con tarjetas reordenables

## Stack

- Flutter 3.x
- Riverpod (estado)
- Isar (base de datos local)
- fl_chart (gráficos)
- Material 3

## Requisitos

- Flutter SDK >= 3.11.0
- Dart >= 3.11.0

## Ejecutar

```bash
flutter pub get
flutter run
```

## Plataformas

- iOS, Android, Web, Windows
